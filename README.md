# Customer Segmentation using K-Means Clustering

## Objective
Segment retail customers based on Annual Income and Spending Score.

## Dataset
Mall Customers Dataset

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Steps
- Data Loading
- Data Exploration
- Feature Scaling
- Elbow Method
- K-Means Clustering
- Visualization
- Cluster Analysis

## Results
Successfully segmented customers into 5 distinct groups based on purchasing behavior.
